## LinkedList

```
LinkedList<Integer> list = new LinkedList<>();  //declartion

list.addFirst(5)  //add at beginning
list.addlast(20)  //add at end
list.get(1);  //get a element at index
list.getFirst();  //Get a first element
list.getLast();  //Get a Last element

list.remove(0); //Delete
list.removeFirst();  //Delete first element
list.removeLast();  //Delete Last element

list.size() //Get a length or size
```

## HashSet
Store unique elements (duplication are not allowed)
```
HashSet<Integer> set = new HashSet<>();   // Declaration

set.add(10);          // Insert
set.remove(10);       // Delete
set.contains(10);     // Check if element exists
set.size();           // Number of elements
set.isEmpty();        // Check if empty
set.clear();          // Remove all elements

for(Integer num : set){
    System.out.println(num);   // Traverse
}
```

## Hashmap
Store a key-value pairs (keys are unique)
```
HashMap<Integer, String> map = new HashMap<>();   // Declaration

map.put(1, "Rahul");          // Insert
map.put(1, "Amit");           // Update value for existing key

map.get(1);                   // Get value by key
map.remove(1);                // Delete by key

map.containsKey(1);           // Check if key exists
map.containsValue("Rahul");   // Check if value exists

map.size();                   // Number of key-value pairs
map.isEmpty();                // Check if empty
map.clear();                  // Remove all entries

map.keySet();                 // Get all keys
map.values();                 // Get all values
map.entrySet();               // Get all key-value pairs

// Traverse using keys
for(Integer key : map.keySet()){
    System.out.println(key + " " + map.get(key));
}

// Traverse using entries (Recommended)
for(Map.Entry<Integer, String> entry : map.entrySet()){
    System.out.println(entry.getKey() + " " + entry.getValue());
}
```
